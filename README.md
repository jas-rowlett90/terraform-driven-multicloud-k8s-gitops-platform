# Terraform-Driven Multi-Cloud Kubernetes Reliability & GitOps Platform

## Overview

This project demonstrates a production-inspired DevOps platform using:

- Terraform
- Kubernetes
- ArgoCD
- GitOps
- Prometheus
- Grafana
- Python Automation
- Azure AKS Deployment Patterns
- AWS EKS Deployment Patterns

## Project Status

🚧 In Development

## Objectives

- Provision infrastructure using Terraform
- Deploy applications using GitOps
- Monitor workloads with Prometheus and Grafana
- Automate reliability validation with Python
- Demonstrate multi-cloud Kubernetes deployment patterns
